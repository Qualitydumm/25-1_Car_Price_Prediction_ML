# 중고차 가격 예측 프로젝트

## 📌 프로젝트 개요

본 프로젝트는 중고차 데이터에 기반하여 차량의 다양한 특성(연식, 주행거리, 연료 타입 등)으로부터 **중고 차량 가격을 예측하는 회귀 문제**를 다룹니다. 데이터 전처리, 탐색적 분석, 모델 학습 및 성능 비교를 통해 최종적으로 가장 우수한 모델을 선정하였습니다.
주요 목표는 **예측 성능과 해석 가능성의 균형**을 갖춘 가격 예측 모델 구축입니다. 




---

## 📂 데이터 전처리

### 1) Levy 결측치 처리

* 가격 대비 비정상적으로 높은 Levy 값은 결측치로 처리
* Model, Prod.Year, Engine volume 등과의 관계를 기반으로 **그룹 중앙값 또는 회귀 대체** 적용 

### 2) Mileage 처리

* 5억 km 이상 값 제거
* 로그 변환 + IQR 방식 이상치 제거
* 일부 모델에서는 연간 주행 가능 거리(약 100,000km) 기준으로 상한 보정 진행 

### 3) Price 처리

* 이상치 제거 후 로그 변환
* IQR 기반 정규화 처리 

### 4) 범주형 변수 처리

* Model-Fuel type 기반 Trim 생성 후 원-핫 인코딩
* Category, Fuel type, Gear box type 등 One-hot 인코딩 진행

## 🔍 주요 인사이트

모델별 중요 변수 분석 결과 중고차 가격에 영향을 크게 미친 요소는 다음과 같습니다:

| 순위 | 변수            | 해석                   |   |
| -- | ------------- | -------------------- | - |
| 1  | Engine volume | 배기량 증가에 따른 가격 상승 경향  |   |
| 2  | Mileage       | 누적 주행거리 증가에 따른 가격 하락 |   |

| 4  | Gear box type | 자동/수동에 따른 시장 가격 차이   |   |
| 5  | Drive wheels  | 구동 방식에 따른 가격 차이      |   |
